# Contributing

Contributions are welcome for:

- New AI security controls
- Better framework mappings
- Detection engineering examples
- Guardrail implementation examples
- Vendor review questions
- AI incident response improvements
- Documentation fixes

Please avoid copying proprietary control text from paid standards, internal company policies, or vendor documentation.

Suggested contribution format:

1. Problem or gap
2. Proposed control or improvement
3. Evidence or public reference
4. Suggested workbook tab/section
