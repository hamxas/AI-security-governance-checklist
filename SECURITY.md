# Security Policy

## Reporting Issues

If you find a security issue, privacy issue, or incorrect control recommendation in this checklist, please open a GitHub issue or contact the maintainer privately.

## Scope

This repository contains an assessment workbook and documentation. It does not contain production software or an AI gateway implementation.

## Disclaimer

The checklist is for educational and advisory use. Organisations should validate controls against their own legal, privacy, regulatory, and security requirements.
